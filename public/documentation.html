Welcome to the myFlix API

Request URL HTTP Method Request Body Data Format in JSON Response Data Format in JSON
Get a list of all movies with details. /movies GET None A JSON array containing movie data. Example: [ { "Title": "The Shawshank Redemption", "Description": "Two imprisoned men bond over several years...", "Genre": "Drama", "Director": "Frank Darabont", "ImageURL": "https://example.com/shawshank_redemption.jpg", "Featured": false }, // ... (other movie objects) ]
Get data about a movie by title. /movies/:title GET None A JSON object holding data about a single movie. Example: { Title: 'Pulp Fiction', Description: "A nonlinear narrative weaves together the lives of various characters, including hitmen, a boxer, and a gangster's wife, in this iconic Quentin Tarantino film.", Genre:'Drama', Description: 'A Quentin Tarantino crime drama with a unique narrative style.' Director:'Quentin Tarantino', ImageURL: 'https://example.com/pulp_fiction.jpg', Featured: false }
Get data about a genre by genre's name /movies/genre/:genreName GET None A JSON object holding data about a genre. Example: { "Name": "War", "Description": "A gripping war film that delves into the horrors of the Vietnam War." }
Get data about a Director by director's name /movies/directors/:directorName GET None A JSON object holding data about a director. Example: { Director: 'Stanley Kubrick', Bio: 'Stanley Kubrick was a visionary filmmaker with a unique cinematic style.', Birth: 1928 ImageURL: 'https://example.com/full_metal_jacket.jpg', Featured: false }
Add a new user /users POST A JSON object holding user data. Example: { " Username: 'updated', Password: 'anez123', Email: 'anezbjork@gmail.com', Birthday: "2000-07-10" } A JSON object holding user data, including a new ID. Example: { "Username": "darabjork", "Password": "darin123", "Email": "daraupdated@gmail.com", "Birthday": "1997-09-05", "ID": "3f4b1136-c60f-40de-9ca8-77de8021c92c" }
Update user Info /users/:Username PUT A JSON object holding the updated user data. Example: { "Username": "darabjork", "Password": "newpassword", "Email": "newemail@gmail.com", "Birthday": "1997-09-05" } A JSON object holding the updated user data. Example: { "Username": "darabjork", "Password": "newpassword", "Email": "newemail@gmail.com", "Birthday": "1997-09-05" }
Add a movie to a user's list of favorites: /users/:Username/movies/:MovieID POST None A JSON object holding user data with the added movie to their favorites. Example: { "Username": "darabjork", "Password": "newpassword", "Email": "newemail@gmail.com", "Birthday": "1997-09-05", "FavoriteMovies": ["Twilight"] }
Remove a movie from a user's list of favorites /users/:Username/movies/:MovieID DELETE None A JSON object holding user data with the removed movie from their favorites. Example: { "Username": "darabjork", "Password": "newpassword", "Email": "newemail@gmail.com", "Birthday": "1997-09-05", "FavoriteMovies": [] }
Delete user registration /users/:Username DELETE A text indicating that the user has been removed from the system. A JSON object confirming the user deletion. Example: { "message": "User darabjork has been deleted." }
